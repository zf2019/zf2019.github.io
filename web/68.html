网站建设教程简单讲解之[五]网站建设要注意的事项及其它细节

1、网站建设中要遵守和注意的事项

（1）不论使用的是哪国主机空间，一定要遵守空间所在国的法律和中国的法律，否则被墙、被K、被闭、被封杀是一定的（必须的）；

（2）仔细查看和遵守空间提供商的规定，不然网站被关闭还不知道是怎么回事，一般空间商网站都会有详细说明；

（3）不要在空间里放病毒文件、不要在网页上放置恶意代码或挂马；

（4）不要做一些网址导航类或采集类垃级站，这对你的学习一点用处都没有，更无任何发展空间，做这些垃级站纯粹是浪费时间；

（5）一定要养成定期备份网站的习惯，否则当有一天发现网站被闭了（或被黑了），你哭都来不及；

（6）如果是低配置的空间最好不要提供视频类或下载类服务，因为这样做会严重占用空间带宽增加服务器的压力，建议有这方面需要的朋友可以使用第三方提供的服务；

（7）其它站长应遵守和注意的事项（这一条也很重要，因为我发现很多法律法规最后都加上这一条）。

注：前三条很重要一定要紧记

2、建站其它细节讲解

（1）相对路径和绝对路径的理解

有很多人对这两个概念不是很清楚，这会出现在编辑网页的时候里面的内容显示正确而上传到空间里就显示不正确的情况（比如图片无法显示等），下面以实例讲解这两种路径的区别。

1）相对路径(Relative Path)的表示方法

大家都知道网站中包含了很多目录（文件夹），那么如何表示同级目录的文件呢？ 实例：2.html和3.html在同一个文件夹www下， 如果2.html链接到3.html，可以在2.html中这样写:

<a href=“3.html”>链接到同目录中的3.html</a>

如何表示上级目录的文件 实例：1.html是2.html和3.html的上级目录中的文件，如果2.html或3.html链接到1.html，可以在2.html或3.html中这样写

<a href=“../1.html”>链接到上级目录中的1.html</a>

../ 代表一级上级目录(间隔一个目录) ../../代表二级上级目录(间隔两个目录)，4.html是2.html或3.html下级目录中second的文件，如果4.html链接到1.html，可以在4.html中这样写

<a href=“../../1.html”>链接到上级目录的上级目录中的1.html</a>

如何表示下级目录的文件

实例：2.html和3.html是1.html的下级目录first中的文件，如果在1.html中链接到2.html， 可以在1.html中这样写

<a href=“first/2.html”>链接到下级目录(first)中的2.html</a>

如果在1.html中链接到4.html，可以在1.html中这样写

<a href=“first/second/4.html”>链接到下级目录(first/second/)中的4.html</a>

注：相对路径的好处是网站内容方便移植(比如把网站做为另一个网站的一部份，直接移动到另外一个网站中的文件夹中就可以了，不需要再去频繁的更改路径。)，缺点表示路径比较繁琐，需根据所处的位置不同而更改路径。

2）绝对路径(Absolute Path)

是带有网址的路径，比如你有一个域名localhost/wordpress，和一个网站空间，上面的四个文件就可以这么表示。

<a href=“http://localhost/wordpress/www/1.html">链接到1.html</a> 
<a href=“http://localhost/wordpress/www/first/2.html">链接到2.html</a> 
<a href=“http://localhost/wordpress/www/first/3.html">链接到3.html</a> 
<a href=“http://localhost/wordpress/www/first/second/4.html">链接到4.html</a>

注：使用绝对路径的好处是表示路径比较简单，不用再去根据目录的不同而更改路径，缺点是程序不容易移植。

（2）使用网站计数器统计网站的访问量

网站建成后站长想知道网站的具体流量数据的话，就必需使用网站流量统计工具了。流量统计可以使站长了解到有多少人访问过网站，什么时候访问的，以及具体的来源页面等，让站长对自己的网站做到心中有数，是站长们最重要的工具之一。流量统计软件(网站计数器)可以自己编写，但是很耗时，而且很难做完善，网上有很多免费的流量统计工具，建议大家使用。

几个常用的流量统计工具：

百度统计 – Baidu的统计系统。

我要啦免费统计 – 一个不错强大的流量统计工具。

中国站长联盟 – 一个不错强大的流量统计工具。

这些流量统计工具使用起来都很简单，一般都是在自己网站的首页加上他们提供的统计代码就可以了，如果不会用可以通过相关网站的教程学习。流量统计是站长们最重要的工具之一也是做SEO的重要工具之一，它可以使你知道网站的访问量，访问量的来源（比如是google搜索来的，还是别的链接上来的），搜索关键字的使用量，浏览者所在地等重要信息，站长们可以通过流量统计的数据，分析网站访问者的喜好，从而改变网站发展的策略。

（3）关于备案

在中国建网站是必需强制性备案的，当然如果你的网站空间在国外也可以不备案，我在这里只所以提到这个问题是想说明一点，不论你的服务器是在哪国，你网站针对的目标人群一定90%是中国人吧（为什么？因为你正在看我的教程），既然是针对中国人群就算你不备案也会受到限制，所以就算没有备案也一定要遵守中国的相关规定，不然你的网站就只能给老外看了。其实我也很烦备案，因为我有切身的体会，如果走正常程序的话你一个月内能备下来就算是很快的了，我曾经备案一个网站花了三个月时间，修改了N次才提交成功（如果找黄牛的话听说最多三天搞定，网上有很多人从事这个职业），不过现在在国内买空间的话空间商一般会有免费提供备案的服务。如果你想了解备案的具体流程，可以到网上搜索相关的教程，我在这里就不多说了。

（4）关于版权与共享

如果是做博客站的话，都会遇到版权与共享的问题，现在国内加大了对知识产权的保护力度，当然在网上的自由度就开放多了，有很多信息是可以共享发布的。我的个人建议是：在转载别人文章的时候一定要看他们相关的版权信息（一般是在文章底部或网站底部），如果有可以转载的声明才可以转载但是一定要加上文章转载处信息如“转载自某某网站，原文网址”，这是对别人劳动成果的充份尊重，当然最好是多发自己的原创文章让别人转载。 在自己的网站中最好也加上版权信息，个人博客推荐使用creativecommons.org提供的License共享协议，其中有很多共享条目可以根据自己的实际情况任意组合，然后把链接放到网站上提醒别人遵守。
